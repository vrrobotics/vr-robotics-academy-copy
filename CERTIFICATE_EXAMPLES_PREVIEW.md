## How to Add These Certificates

### Option 1: Admin Panel (Easiest)
1. Go to your admin panel
2. Add the CertificateManagement component
3. Click "Add Certificates" button
4. Done! Page reloads with new certificates

### Option 2: Browser Console (Quickest)
Run in browser console (F12):
```javascript
await window.certificateUtils.quickAddCertificates();
```

### Option 3: Service Function
```typescript
import { addCertificateExamples } from '@/services/certificateService';
await addCertificateExamples();
```
